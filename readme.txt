Environment requirement：
1、Jdk1.8 or later.
2、maven3.6.3 or later.

Execution step：
1、Enter the project directory,run 'mvn spring-boot:run' in the command line.
2、Follow the instructions in the command line.

Assumptions:
1、Make sure your computer has connected to the internet,otherwise,it won't get currency exchange.
2、the input of currency must be 3 uppercase letter code,otherwise,it will print an error message and exit.
3、the input of amount must be a number,otherwise,it will print an error message and exit.
4、the currency and amount must be split by one space,otherwise,it will print an error message and exit.

Remark:
The 'input.txt' file is an example of formatted input.
